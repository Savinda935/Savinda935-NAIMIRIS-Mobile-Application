# NAIMIRIS Mobile (Expo)

React Native (Expo) structure for the NAIMIRIS smart farming app.

## Structure

- src/config: theme and constants
- src/navigation: app navigation
- src/screens: UI screens for each module
- src/features: domain logic per stage
- src/services: API, AI, IoT, storage stubs
- src/state: app-wide state
- src/components: reusable UI

## Run

1. npm install
2. npx expo start

## Notes

- AI and IoT services are placeholders for proposal-driven integration.
- Logic includes budget-aware planning, ROI, stage-wise guidance, and subscription tiers.
